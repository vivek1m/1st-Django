About This App

Django Calci App is a simple and elegant calculator built with Django and Bootstrap 5. It provides a responsive interface for quick and safe evaluations of mathematical expressions.

Instructions

You can enter Python-style expressions like:

  • 2 + 3 → Addition
  • 7 - 2 → Subtraction
  • 4 * 5 → Multiplication
  • 10 / 2 → Division
  • 10 % 3 → Modulus
  • 2 ** 3 → Power

⚠️ Only valid Python arithmetic expressions are allowed. Avoid using alphabets or unsafe inputs.
